Africa Country Comparison Dashboard

Interactive map and comparison view for African countries, combining World Bank indicators with Freedom House and DSP V-Dem digital-society measures for 2025 where provided.

Country table
Map includes 54 African Union member states. Somaliland and Zanzibar are excluded from the geographic dashboard because they are distinct V-Dem coding units but not standard sovereign map features in this build.
This version uses plain JavaScript with embedded data and explicit DOM references to maximize preview compatibility.
